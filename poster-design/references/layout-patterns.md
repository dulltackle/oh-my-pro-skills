# 布局模式库

不是所有海报都用同一种布局。根据内容量和重点选择。

## 模式选择指南

| 内容特点 | 推荐布局 | 说明 |
|---------|---------|------|
| 有主讲人照片 | **A: 照片主角型** | 照片占 45%+，标题在上方 |
| 无照片/纯文字 | **B: 居中优雅型** | 所有内容居中，大标题+大留白 |
| 信息量大 | **C: 分割杂志型** | 左右分栏或上下分区 |
| 极简信息 | **D: 全幅冲击型** | 超大标题 + 最少文字 |

## Layout A：照片主角型（默认推荐）

```
┌──────────────────────┐
│ Brand · Date         │ ← 顶栏
├──────────────────────┤
│ ★ 大标题（渐变）     │ ← 记忆点
│   副标题             │
│   ─ ● ─ ○           │
├────┬─────────────────┤
│姓名│                 │
│拼音│   [ 主讲人照片 ] │ ← 视觉锚点
│头衔│   (带框架装饰)   │
│竖排│                 │
├────┴─────────────────┤
│ [标签] [标签] [标签]  │
├──────────────────────┤
│ ┌──────────────────┐ │
│ │ □ 讲座内容       │ │ ← 内容卡
│ │ ✓ item           │ │
│ └──────────────────┘ │
├──────────────────────┤
│ 时间 │ 地点 │ 报名   │ ← Info bar
├──────────────────────┤
│ 四列适合人群网格      │
├══════════════════════┤
│ ░░ CTA 大按钮区块 ░░ │ ← 视觉焦点 #2
├──────────────────────┤
│ ―― 机构名 ――        │ ← Footer
└──────────────────────┘
```

**适用**：讲座、课程、培训、有主讲人的活动
**实现**：flex row 布局，左侧人物信息(240px) + 右侧照片(500px)

## Layout B：居中优雅型

```
┌──────────────────────┐
│       Brand          │
├──────────────────────┤
│                      │
│    ★★★ 大标题 ★★★   │ ← 居中，超大
│      副标题          │
│                      │
│   ─── ● ───          │
│                      │
│  （可选小图/装饰）    │
│                      │
│ ┌──────────────────┐ │
│ │    核心内容       │ │ ← 居中卡片
│ └──────────────────┘ │
│                      │
│  时间 · 地点 · 报名  │
│                      │
│ ░░░ CTA 按钮 ░░░     │
│                      │
│     ―― Footer ――     │
└──────────────────────┘
```

**适用**：无照片的活动、发布会、简约风格
**特点**：大量留白、所有元素居中、衬线大标题

## Layout C：分割杂志型

```
┌──────────────────────┐
│ Brand                │
├──────────┬───────────┤
│          │           │
│  大标题  │  [照片]   │ ← 左右分栏
│  副标题  │           │
│          │           │
├──────────┴───────────┤
│ 标签行               │
├──────────────────────┤
│ ┌────┐ ┌────┐       │
│ │卡1 │ │卡2 │       │ ← 多卡片并排
│ └────┘ └────┘       │
├══════════════════════┤
│ CTA (全宽)            │
├──────────────────────┤
│ Footer               │
└──────────────────────┘
```

**适用**：信息量大、多主题、需要展示多个亮点
**实现**：grid 两列，左 40% + 右 60%

## Layout D：全幅冲击型

```
┌──────────────────────┐
│                      │
│  ██████████████████  │ ← 超大标题撑满宽度
│  ██████████████████  │
│                      │
│  ――――――  ――――――     │
│                      │
│      一句话描述       │
│                      │
│ ░░░░ CTA ░░░░        │
│                      │
│  小字辅助信息         │
│                      │

```

**适用**：音乐节、创意活动、极简宣传
**特点**：字号极大(120px+)、文字即视觉、极少装饰

## 间距系统（Tailwind scale）

统一使用以下间距值，不要随机取数：

| Token | Value | 用途 |
|-------|-------|------|
| `gap-1` / `p-1` / `m-1` | 4px | 元素内部微调 |
| `gap-2` / `p-2` / `m-2` | 8px | 标签间距、图标间隙 |
| `gap-3` / `p-3` / `m-3` | 12px | 卡片内 padding |
| `gap-4` / `p-4` / `m-4` | 16px | 区块间距 |
| `p-5` / `p-6` | 20px / 24px | 卡片 padding |
| `py-8` / `py-10` | 32px / 40px | Section 间距 |
| `py-12` ~ `py-16` | 48px / 64px | 大区间距 |

**规则**：
- 同级元素用相同间距
- 区块间间距 > 区块内间距（至少 2x）
- 移动端所有间距减半

## F 形浏览模式适配

用户扫读海报时天然遵循 F 形路径：

```
1. 从左上 → 横读到右上（标题行）
2. 稍下移 → 再横扫（副标题/标签）
3. 沿左侧竖直向下（内容列表左侧竖线引导视线！）
4. 底部横扫（Info bar / CTA）
```

**设计启示**：
- 重要信息放**左上**和**左侧**
- 列表的左侧竖线（`.c-list::before`）不只是装饰——它**引导视线**
- CTA 放在视线自然落脚的**中下部偏下**
- 右上角适合放**次要信息**（日期、页码）
