# 海报图片裁剪与处理技巧

> 本文档聚焦**裁剪理论与 CSS 实现**，使画面更协调、更有视觉冲击力。
> **⚠️ 收到图片后、写代码前，必须先完成 `image-checklist.md` 的素材检查清单。**

## §0 素材检查清单（写代码前的硬门槛）

> 这是 2026-04-10 刘丰硕海报翻车后新增的强制步骤。
> 翻车记录：用户提供了透明底 PNG，但被当作黑底图处理了 5 轮才发现。

### Step 1：命令行快速检查

```bash
# 必须执行的第一条命令
file <图片路径>
```

**输出解读：**

| 输出特征 | 含义 | 后续策略 |
|---------|------|---------|
| `8-bit/color RGBA` | **透明底 PNG** | 最佳素材！用 `object-fit:contain` + `drop-shadow` 直接融入 |
| `8-bit/color RGB` | 不透明 PNG | 需判断背景色（白/深/复杂）
| `JPEG image data` | JPEG（必然不透明） | 同上 |
| `width x height` | 实际像素尺寸 | 判断方向和比例 |

### Step 2：方向与比例判断

根据宽高比判断：

| 比例范围 | 方向 | 推荐容器策略 |
|---------|------|-------------|
| 宽 < 高 (如 922×1344) | 竖版 | 天然适合 Layout A 右侧主角区 |
| 宽 ≈ 高 (±15%) | 近似方形 | 缩小宽度到 40%，左侧信息区加宽 |
| 宽 > 高 (如 16:9) | 横版 | 放标题下方横贯全宽，或裁切为方形 |

### Step 3：背景类型判断（仅非透明底图片需要）

**用视觉能力查看图片，判断背景类型：**

| 背景类型 | 特征 | 融入策略 | 详见章节 |
|---------|------|---------|---------|
| 纯白/浅色 | 证件照、写真 | 圆角+内阴影+渐变蒙版 或 建议用户提供抠图版 | §五 白底照融入 |
| 纯深色 | 艺术照、棚拍 | drop-shadow + 微光晕即可自然融入 | §四 CSS 处理方式 |
| 复杂背景 | 环境/场景照 | 加实心边框/圆角/投影做隔离 | §六 进阶处理 |
| **透明 (RGBA)** | 抠图 PNG | `object-fit:contain` + `drop-shadow` + `overflow:visible` | §五 Knockout |

### Step 4：记录素材档案

写代码前在脑中（或注释中）确认以下信息：

```
格式: PNG(RGBA) / JPEG / PNG(RGB)
尺寸: W x H px
方向: 竖 / 横 / 方
背景: 透明 / 白 / 深 / 复杂
容器策略: contain+drop-shadow / cover+圆角 / cover+边框过渡
object-position: center XX%
```

**⚠️ 不完成此清单不写第一行 CSS。**

## 一、裁剪的 7 种基本方式（McWade 分类）

在海报中放入一张图时，先问自己：我要用哪种裁剪？

| # | 方式 | 做法 | 效果 | 适用场景 |
|---|------|------|------|---------|
| 1 | **Hard Crop 硬裁切** | 直接切割图片边缘，改变比例 | 干净利落，为文字腾出空间 | 最常用，需要精确控制边界时 |
| 2 | **Soft Crop 柔化裁切** | 边缘做羽化/渐变淡出 | 自然过渡，模拟人眼 peripheral vision | 需要图片与背景融合时 |
| 3 | **Split Crop 分割裁切** | 从图片内部切开一段 | 可拉开间距插入文字 | 创意型海报，需要图文穿插 |
| 4 | **Stickout Crop 突出裁切** | 大部分裁掉，留一小部分伸出 | 打破画框限制，增加立体感和动感 | 需要打破方正感的正式海报 |
| 5 | **Knockout Crop 抠像裁切** | 去掉整个背景，只保留主体 | 完全自由地与页面元素交互 | 人物需要脱离原背景时 |
| 6 | **False Crop 假裁切** | 不真切，用色块遮挡部分图片 | 暗示"还有更多内容"，制造悬念 | 封面、杂志风 |
| 7 | **Shape Crop 形状裁切** | 裁成圆形/不规则形状 | 打破矩形单调，增加趣味性 | 装饰性强、偏活泼的海报 |

**对 HTML/CSS 海报的实现映射：**

| 裁剪方式 | CSS 实现 |
|---------|---------|
| Hard Crop | `object-fit: cover` + 固定宽高容器 |
| Soft Crop | `mask-image: linear-gradient()` 边缘渐变，或 `-webkit-mask` |
| Split Crop | 两段 `img` 或 `clip-path` + 间距 |
| Stickout | 容器 `overflow: visible` + 图片超出容器范围 |
| Knockout | PNG 透明底或 CSS `mix-blend-mode` |
| False Crop | 色块 `position: absolute` 叠加在图片上 |
| Shape Crop | `border-radius: 50%` 或 `clip-path: circle/polygon()` |

## 二、人像裁切的黄金法则

### 法则 1：不切关节（最核心！）

> **"If it bends, don't crop it."** — Digital Photography School

人体关节是**绝对禁区**。切了会给人"被截断""断肢"的不适感。

```
❌ 绝对不要切的位置：
   脖子、手腕、手肘、膝盖、脚踝、手指、下巴

✅ 可以安全裁切的位置：
   前臂中段（肘到手腕之间）
   大腿中段（臀到膝之间）
   腰部
   额头上方（发际线以上）
   肩膀以上（保留部分肩部，避免"砍头"感）
```

### 法则 2：眼睛在上三分之一

人像裁切后，**眼睛应位于画面上 1/3 线附近或更高位置**。眼睛太低 = 画面呆滞、缺乏生命力。

### 法则 3：头顶留空间（不顶天）

人物头顶不能贴着画布顶部——要留呼吸空间。否则画面压抑、拥挤。

### 法则 4：下巴要完整

裁切下巴会让脸看起来方、笨重。即使要做特写，也保留完整的下巴线条。

### 法则 5：裁切要有意图感

每次裁切都要看起来是**故意的**，而不是失误。如果不确定，做两个版本对比。

## 三、人像构图的 6 种取景范围

来自人像摄影口诀："竖构图齐胸处；正侧背转头部；退一步切臀部；进一步肩到颅；全身图拍清楚；不顶天巧裁处"

| 类型 | 裁切位置 | 海报中的用途 | 气质 |
|------|---------|-------------|------|
| **大头照/特写** | 肩以上（必须带一点肩膀，别切脖子） | 强调面部表情、个人品牌 | 亲切、直接、有冲击力 |
| **标准照/胸像** | 胸部上下 | 正式介绍、权威形象 | 专业、正式、可信 |
| **半身照** | 臀部/腰部 | 展示姿态和着装 | 平衡、自然 |
| **七分身** | 膝盖以上 | 展示整体造型 | 舒展、大气 |
| **九分身/小全身** | 小腿以下（不露脚） | 全身展示但省空间 | 完整、有气势 |
| **大全身** | 完整包含脚 | 需要展示服装全貌 | 完整、真实 |

**海报实践建议：**
- 人物介绍海报 → **标准照或半身照**最佳（兼顾面部辨识度和身体语言）
- 特邀嘉宾/演讲者 → **胸像以上**（突出面部，配合头衔文字）
- 避免：用全身照硬塞进一个小容器 → 必然导致面部太小

## 四、海报中人物图片的 6 种进阶处理方式

来自站酷「人物海报设计中 6 种常见处理方式」+ 知乎 18 个海报技巧：

### 1. 文案点缀型

**适用**：照片本身构图好、光影好、人物个性突出
**做法**：几乎不加修饰，只配上简洁有力的字体
**关键**：字体风格要与照片氛围匹配（旧照片配复古体、正式照配衬线体）
**HTML 技巧**：文字层用 `position: absolute` 叠加在照片区域旁或上方

### 2. 点线面强化型

**适用**：原图冲击力不够强，需要增强设计感
**做法**：在照片上加几何元素（线条、色块、边框），区分字体层级但不弱化人物
**关键**：几何元素是辅助，不能抢过人物的视觉焦点
**HTML 技巧**：SVG overlay 或 CSS 伪元素画线框

### 3. 插画/涂鸦叠加型

**适用**：需要增加创意效果和情绪张力
**做法**：在人物身上或周围添加插画/涂鸦元素
**注意**：涂鸦风格要与整体调性一致

### 4. 文字与图像融合型

**适用**：需要大标题但不希望弱化人物
**做法**：文字与人物画面相互叠加/穿插，文字成为画面的一部分
**HTML 技巧**：`mix-blend-mode: multiply/screen` 或文字层 `z-index` 交错

### 5. 拼贴切割型

**适用**：多人海报或有多个角度的照片
**做法**：分割/拼贴组合，保持色调一致
**HTML 技巧**：CSS Grid 多图布局 + 不等分栏

### 6. 特效合成型

**适用**：需要艺术化表达
**做法**：双重曝光、渐变融合、故障艺术等
**HTML 技巧**：`mix-blend-mode` 系列 + CSS 渐变 + `clip-path`

## 五、提升人物图片质感的 4 个手法

来自「即时设计·高级感人物海报」：

### 1. 光影效果
- 用合理的光影处理突出人物立体感
- 模拟自然光增强轮廓，使人物从背景脱颖而出
- CSS：`drop-shadow` + 径向渐变光晕背景

### 2. 边缘柔化（Soft Edge）
- 图片边缘不做生硬矩形，而是微渐变淡出
- 让图片更自然地融入海报背景
- CSS：`-webkit-mask-image: radial-gradient()` 或 `mask-border`

### 3. 细节润色
- 如果条件允许（有编辑权限）：皮肤质感优化、面部光影柔化
- 在纯 CSS 海报中可通过 `filter: contrast/saturate/brightness` 微调

### 4. 背景分离
- 复杂背景的人物图 → 加实心边框/圆角/投影做隔离
- 纯色/浅色背景的人物图 → 可做边缘渐变融合
- CSS：复杂背景用 `padding + border-radius + box-shadow`; 简单背景可尝试 `mask-image` 渐变

## 六、裁剪决策流程图

拿到一张人物照片后：

```
Step 1: 确认照片比例和方向
   ├─ 竖版(3:4+) → 天然适合竖版海报右侧主角区
   ├─ 近似方形(4:5~1:1) → 缩小宽度到40%，左侧信息区加宽
   ├─ 横版(16:9+) → 考虑横放标题下方，或裁切为方形/竖版
   └─ 记录：实际尺寸、主体位置、背景复杂度

Step 2: 选择取景范围
   ├─ 需要强调面部辨识度 → 特写/胸像
   ├─ 需要展示权威/正式感 → 标准照
   ├─ 需要展示姿态/着装 → 半身/七分身
   └─ 照片质量一般 → 裁近一些（特写）隐藏瑕疵

Step 3: 选择裁剪方式
   ├─ 需要干净利落 → Hard Crop (object-fit: cover)
   ├─ 需要与背景融合 → Soft Crop (mask 渐变)
   ├─ 需要突破画框 → Stickout (overflow: visible)
   ├─ 需要去掉糟糕背景 → Knockout (PNG/mix-blend-mode)
   └─ 需要创意效果 → Shape/Split (clip-path)

Step 4: 安全检查
   ├─ 关节部位是否被切？→ 是 → 调整裁切位置
   ├─ 眼睛是否在上1/3？→ 否 → 上移视线或重新裁切
   ├─ 下巴是否完整？→ 否 → 降低裁切线
   ├─ 头顶是否有呼吸空间？→ 否 → 增加上方 padding
   └─ 裁切看起来是有意为之还是意外？→ 意外 → 调整到有意为止

Step 5: 与其他元素的协调
   ├─ 照片与文字是否抢焦点？→ 是 → 降低照片饱和度或缩小尺寸
   ├─ 照片边缘与背景是否冲突？→ 是 → 加 border/glow/shadow
   └─ 照片在整体空间预算中占比是否 ≤45%？→ 否 → 缩小照片
```

## 七、CSS 代码片段速查

### Soft Crop（边缘渐变淡出）

```css
.photo-soft-edge {
  -webkit-mask-image: linear-gradient(
    to bottom,
    black 85%,
    transparent 100%
  );
  /* 或四边都淡出 */
  -webkit-mask-image: radial-gradient(
    ellipse at center,
    black 70%,
    transparent 100%
  );
}
```

### Stickout Crop（突破容器）

```css
.photo-stickout {
  width: 90%;        /* 比容器窄 */
  margin-left: auto; /* 偏右 */
  margin-right: -10%; /* 右侧溢出容器 */
  border-radius: 4px;
  box-shadow: 0 20px 60px rgba(0,0,0,.15);
}
/* 父容器需 overflow: visible */
```

### Shape Crop（圆形/异形）

```css
.photo-circle {
  clip-path: circle(45% at 50% 50%);
  /* 或多边形 */
  clip-path: polygon(10% 0%, 90% 0%, 100% 90%, 0% 100%);
}
```

### Knockout 效果（混合模式去背景）

```css
/* 适用于深色背景上的深色图片 */
.photo-knockout {
  mix-blend-mode: multiply;
  /* 或 screen（适用于亮背景上的亮色图） */
}
```

### Split Crop（分割拉开）

```css
.photo-split-container {
  display: flex;
  gap: 16px;       /* 分割间距，可放文字 */
}
.photo-split-container img {
  flex: 1;
  object-fit: cover;
}
```

## 八、反模式速查

| ❌ 错误 | ✅ 正确 |
|--------|--------|
| 在关节处裁切（脖子/手腕/膝盖/肘部） | 在骨骼中间段裁切（前臂/大腿/腰部） |
| 眼睛在画面下半部分 | 眼睛在上 1/3 线附近 |
| 头顶贴着容器顶 | 头顶留 ≥8% 的呼吸空间 |
| 切掉了下巴 | 保留完整下巴线条 |
| 用 `object-fit: cover` 但没检查裁切结果 | 先确认 object-position 后再定死 |
| 全身照塞进小容器导致面部如豆粒 | 改用胸像/半身裁切 |
| 照片占画布 >50% | 控制在 40-48% |
| 生硬矩形贴在复杂背景上 | 加 border-radius + shadow + glow 做隔离 |
| 同一张图既放 badge 又写正文重复信息 | 只保留一处 |

## 九、白底证件照融入策略（实战经验 2026-04-09）

人物介绍海报最常见的素材是**白底证件照**。这类照片直接贴在暖色/有色背景上会产生强烈的视觉冲突。

### 问题诊断

| 症状 | 原因 | 严重度 |
|------|------|--------|
| 照片像一张纸片贴在海报上 | 白底与背景色差太大 | 🔴 致命 |
| 人物边缘有硬边锯齿 | 没有做边缘柔化 | ⚠️ 严重 |
| 照片区域看起来"浮"在背景上 | 缺少光晕/投影融合 | ⚠️ 严重 |

### 融入方案（按效果从好到差排序）

| 方案 | 效果 | 实现方式 |
|------|------|---------|
| **⭐ 抠图换透明底**（最佳） | 人物自然融入任何背景 | 用图像编辑工具抠图，存为 PNG 透明底 |
| **圆角 + 内嵌阴影 + 渐变蒙版**（次优） | 边缘柔化，减少硬边感 | `border-radius:12px` + `box-shadow:inset 0 0 40px rgba(背景色,.15)` + `::after` 蒙版 |
| **多层光晕包围**（辅助） | 在照片和背景间建立过渡层 | 2-3 层 `radial-gradient` 光斑，颜色取自背景主色和 accent |
| **粗描边隔离**（最低限度） | 用边框框住照片形成"画框"效果 | `border:1px solid rgba(accent,.2)` + 角装饰 |

### CSS 实现参考

```css
.photo-wrap {
  border-radius: 12px;        /* 大圆角 */
  overflow: hidden;
  position: relative;
  box-shadow: 0 30px 70px rgba(0,0,0,.18);  /* 强投影接地 */
}
/* 边缘柔化蒙版 */
.photo-wrap::after {
  content: '';
  position: absolute;
  inset: 0;
  box-shadow: inset 0 0 40px rgba(246,237,231,.15),   /* 匹配背景色的内阴影 */
              inset 0 0 80px rgba(246,237,231,.08);
  pointer-events: none;
  border-radius: 12px;
}
```

**注意：以上 CSS 方案只能改善，不能根本解决。如果用户对质量要求高，应主动建议提供抠图后的透明底素材。**
