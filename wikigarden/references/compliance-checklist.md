# 强制执行协议与检查清单

> ⚠️ 每次触发 wikigarden 时，必须在执行前对照 Pre-flight 清单。
> 各操作的 🔒 检查项在对应 references 文件末尾，执行该操作时按需对照。

## 阶段门控（Phase Gate）

执行过程中遇到以下标记时**必须停下来等用户回复**：
- `⏸️ GATE` — 需要用户确认/决策才能继续
- 未经用户明确许可，**不得越过 GATE 继续执行**

## 禁止行为

- 不得以「时间晚」「用户可能不想被打断」为由跳过确认步骤
- 不得自行合并或省略标有 🔒 的步骤
- 如确信某步可跳过，必须**先向用户说明理由并获得许可**

## 🔒 Pre-flight（执行前，全部必做）

- [ ] 已读取并理解当前任务属于 Ingest / Query / Lint 哪种操作
- [ ] 如为 Ingest：已与用户确认主题边界、素材来源、数量预期（Ingest Step 1）
- [ ] 已读取对应操作的 references 文件（ingest.md / query.md / maintenance.md）
- [ ] 已确认 provider 配置和凭证可用

## 执行后自检

- [ ] 对照所执行操作的 🔒 检查项，逐条确认已完成
- [ ] 如有未完成项，向用户说明原因并获得明确许可
- [ ] 输出简要执行报告：做了什么 / 跳过了什么 / 为什么
